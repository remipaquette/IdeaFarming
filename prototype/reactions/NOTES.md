# Prototype Notes — Idea Reactions

**Question being answered:** How should employees express support/engagement for an Idea?

**Prototype:** `prototype/reactions/index.html`  
**Run:** Open `prototype/reactions/index.html` in a browser (no install needed). Use the bottom bar to switch between variations.

---

## Variations

| Variation | Description |
|---|---|
| **A** | Single upvote button with count. Clean, sorts easily, feeds a ranking algorithm. |
| **B** | Four fixed reactions (👍 Support / 💡 Inspiring / 🔧 Actionable / 🔥 Urgent) with per-reaction counts. More signal per interaction. |
| **C** | Upvote count (for ranking) + reactions side by side. Maximum signal, busier UI. |
